# CodeVision 

CodeVision is a web application that allows users to upload fragments of their coding projects online. Users upload a video showcasing their projects and its functionalities while also being able to interact with other users' projects through likes + connections. Users can choose to see what they are shown by filtering out projects on different categories. Included with Google Log-In System.
